# Procter & Gamble (procter-and-gamble)
Procter & Gamble is a multinational consumer goods corporation and one of the largest in the world. P&G operates an API Marketplace at developer.pg.com that provides partners, suppliers, and developers with access to APIs for integrating with P&G's supply chain, product data, and business operations.
